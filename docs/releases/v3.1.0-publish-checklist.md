# v3.1.0 Publish Checklist

## 1. Final quality gate
```bash
ruff check .
black --check .
mypy
python -m unittest discover -s tests -v
```

## 2. Contract and stack smoke checks
```bash
python generate_portfolio_schema.py
python validate_evaluation_contract.py --input portfolio_evaluation_local.json
python tests/ci_smoke_stack_profiles.py --scenario all
```

## 3. Confirm release docs
- `CHANGELOG.md`
- `docs/migration-v3_0-to-v3_1.md`
- `docs/releases/v3.1.0-release-notes.md`

## 4. Tag and push
```bash
git add .
git commit -m "release: v3.1.0"
git tag -a v3.1.0 -m "Portfolio Fit v3.1.0"
git push origin main
git push origin v3.1.0
```

## 5. GitHub Release
- Create release from tag `v3.1.0`.
- Use content from `docs/releases/v3.1.0-release-notes.md`.
- Attach optional calibration/benchmark artifacts if needed.
