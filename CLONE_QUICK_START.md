- ✅ Ничего не теряется

## 📥 Что Происходит?

1. **Получение данных**: Скрипт запрашивает список репо через GitHub API (публичное)
2. **Создание папки**: Создаёт папку `~/github/` если её нет
3. **Клонирование**: Для каждого репо выполняет `git clone <url>`
4. **Статистика**: Показывает результаты и размеры

## ⚠️ Важные Замечания

### Сколько Это Займёт?

Зависит от:
- Количества репозиториев (~30?)
- Их размера (~5-20 MB в среднем)
- Скорости интернета

**Примерное время:** 5-15 минут для 30 репо среднего размера

### Сколько Места На Диске?

Примерно **250-500 MB** для всех репо (если у вас ~30 средних проектов).

Проверить:
```bash
du -sh ~/github
```

### Если Репо Очень Большой?

Скрипт установит таймаут 120 секунд. Если репо больше - он будет пропущен.

Чтобы клонировать такой репо вручную:
```bash
git clone https://github.com/ergon73/имя_репо.git ~/github/имя_репо
```

### Если Интернет Нестабильный?

Просто запустите скрипт снова - он продолжит с того же места и добавит недостающие репо.

## 🆘 Если Что-то Не Работает

### Ошибка: "git not found"

```bash
# Установите git
brew install git  # macOS
# или
sudo apt-get install git  # Linux
```

### Ошибка: "Permission denied"

```bash
# Дайте права на выполнение
chmod +x clone_all_repos.py

# Запустите
python clone_all_repos.py ergon73
```

### Ошибка: "Timeout"

Значит большой репо. Просто запустите скрипт снова - продолжит с того же места.

### Не Уверены Что-то Ли Клонировалось?
